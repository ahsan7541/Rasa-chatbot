### Important files
For training : nlu.yml, storeis.yml, domain.yml <br>
For UI       : credentials.yml, index.html

### Important commands
conda deactivate <br>
conda activate rasaenv <br>
pip install rasa <br>
rasa init 

rasa train <br>
rasa shell <br>
rasa run -m models --enable-api --cors "*"
